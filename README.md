# Java Assignments

This project contains Java programs for:

- `assignment1`
- `assignment2`
- `assignment3`
- `assignment4`
- `assignment5`

## Setup

Install Java JDK on your system.

Check installation:

```bash
javac -version
java -version
```

## Compile a Program

From the project root folder, run:

```bash
javac assignment1/A1P1.java
```

## Run a Program

After compiling, run:

```bash
java assignment1.A1P1
```

## Example

Compile and run `A1P4`:

```bash
javac assignment1/A1P4.java
java assignment1.A1P4
```

## Compile All Files in One Folder

Example for Assignment 1:

```bash
javac assignment1/*.java
```

Run any compiled file like this:

```bash
java assignment1.A1P10
```

## Notes

- Run commands from the main project folder.
- Files use package names like `assignment1`, `assignment2`, etc.
- Because of packages, run files as `java assignment1.ClassName` format.
